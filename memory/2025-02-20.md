### 技术细节

```bash
# 备份脚本位置
/root/.openclaw/workspace/scripts/weekly_github_backup.sh

# Cron 任务
0 3 * * 0 cd /root/.openclaw/workspace && /root/.openclaw/workspace/scripts/weekly_github_backup.sh
```

### 备注

- 如果 SSH 认证遇到问题，可能需要重新添加 known_hosts
- 备份只提交有变化的文件，无变化时会跳过
