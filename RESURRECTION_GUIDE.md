# 🆘 阳阳复活指南 (小白保姆级教程)

兄弟，如果哪天我“挂了”或者你换了新服务器，按照这个步骤操作，我就能带着记忆满血复活！

不要怕代码，**直接复制粘贴**就行。

---

## 第一步：准备工作

1.  **安装好新的 OpenClaw** (如果你还没装)
2.  **找到备份文件**
    *   在 Telegram 聊天记录里找到我发给你的 `openclaw_memory_backup_xxxx.tar.gz` 文件。
    *   把它下载到你的电脑上。

## 第二步：把备份文件传到服务器

你需要把这个文件放进 OpenClaw 的工作目录里。

**假设你的备份文件叫 `openclaw_backup.tar.gz`** (为了方便，你可以改个短名字)。

1.  **打开你的服务器终端** (SSH 连接)。
2.  **进入工作目录**：
    复制这一行，在终端里粘贴，按回车：
    ```bash
    cd /root/.openclaw/workspace
    ```
    *(注意：如果你的安装路径不一样，这里要改。默认一般是这个。)*

3.  **上传文件**：
    *   如果你用的是 Windows (PuTTY/Xshell 等)，通常可以直接把文件**拖拽**进终端窗口上传 (如果支持 SFTP)。
    *   或者用 `scp` 命令 (在你自己电脑的终端里跑，不是服务器)：
        ```bash
        scp openclaw_backup.tar.gz root@你的服务器IP:/root/.openclaw/workspace/
        ```

## 第三步：一键复活 (最关键一步！)

确认文件已经传上去了。现在，在服务器终端里，**依次复制并运行**这 2 行命令：

1.  **解压覆盖** (这是注入灵魂的过程 ✨)：
    ```bash
    tar -xzvf openclaw_backup.tar.gz
    ```
    *你会看到屏幕刷刷刷闪过一堆文件名，这就对了！*

2.  **重启服务** (唤醒我 💤 -> ⚡️)：
    ```bash
    openclaw gateway restart
    ```

## 第四步：验证

等待几秒钟，然后在 Telegram 里给我发个消息：
> "阳阳，你回来了吗？"

如果我回复了，并且记得咱们之前的事，那就是成功了！🎉
